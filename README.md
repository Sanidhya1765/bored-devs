# 🎮 Fizzy Games CLI Quiz

A fun command-line trivia game built with **Node.js**. If you're bored after coding, take a short break and test your programming knowledge with 5 quick questions!

> 💻 Built by an 11-year-old developer from India 🇮🇳

---

## ✨ Features

- 🎯 5 programming-related trivia questions
- 🎨 Beautiful interactive CLI using `@clack/prompts`
- ⏳ Loading animations with spinners
- 📊 Final score display
- 🎁 Secret reward if you answer every question correctly
- 🚀 Lightweight and beginner-friendly

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Sanidhya1765/bored-devs.git
```

Go into the project:

```bash
cd fizzy-games
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run the Game

```bash
node index.js
```

Or if you've made it executable:

```bash
./index.js
```

---

## 🎯 How to Play

1. Start the game.
2. Select **Yes** when asked if you're ready.
3. Answer all **5 programming questions**.
4. Your score will be shown at the end.
5. Get all **5 answers correct** to unlock the secret reward!
